# PoliticalEngine

Fabric мод для Minecraft — движок войн и политики.

## Возможности

- Создание городов и наций
- Система клейма чанков
- Защита территорий
- Сценарии (загружаемые из YML)
- Интеграция с BlueMap
- Команды для управления

## Требования

- Minecraft 1.20.1+
- Fabric Loader 0.14+
- Fabric API
- Java 17+

## Сборка

./gradlew build

Готовый .jar появится в build/libs/
